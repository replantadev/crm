CRM Plugin para WordPress
========================

Este plugin permite la gestión de clientes y roles comerciales en WordPress.

## Actualizaciones automáticas

El plugin se actualiza automáticamente desde los releases publicados en GitHub:
https://github.com/replantadev/crm

Para habilitar actualizaciones automáticas:
1. Instala las dependencias con Composer en la carpeta del plugin:
   composer install
2. Sube la carpeta `crm-plugin` a tu repositorio de GitHub.
3. Cuando publiques un nuevo release en GitHub, aparecerá la actualización en las instancias de producción.
